## 1.1 다루는내용
- 파이썬으로 데이터를 다루는 다양하고 기본적인 방법을 소개함
- 파이썬 언어 일부와 데이버 분석 문제를 효율적으로 해결하는데 도움이 되는 몇가지 라이브러리 소개함

### 1.1.1 사용하는 데이터
- 여기서 '데이터'란 구조화된 데이터를 말함
- 각 열의 형식이 문자열, 숫자, 날짜 등으로 서로 다른 표 혹은 스프레시트 형태의 데이터, 관계형 데이터베이스 또는 탭이나 쉼표로 구분된 텍스트 파일 형식으로 저장되는 대부분의 데이터를 포함
- 다차원 배열(행렬)
- 일정하거나 일정하지 않은 간격의 시계열


## 1.2 데이터 분석에 파이썬을 사용하는 이유
다양한 라이브러리의 지원으로 범용 프로그래밍 언어일 뿐만 아니라 과학 계산용으로도 손색이 없기에 데이터 애플리케이션 개발을 위한 최고의 언어임

### 접착제 처럼 쓰는 파이썬 
- 파이썬은 c, c++, 포트란 코드와 통합이 쉬워 과학기술 컴퓨팅 분야에서 성공을 거둠
- 최신 컴퓨팅 환경에서는 선형대수, 최적화, 적분, 고속 푸리에 변환과 기타 알고리즘을 위해 C 라이브러리나 포트란 레거시를 공유

## 1.3 파이썬 필수 라이브러리 

### 1.3.1 넘파이(numpy)
NumPy : Numerical Python의 약자로 자료구조, 알고리즘 등 산술 데이터를 다루는 대부분의 과학 계산 에플리케이션에 이용
- 빠르고 효율적인 다차원 배열 객체 ndarray
- 배열 원소를 다루거나 배열간의 수학 계산을 수행하는 함수
- 디스크로부터 배열 기반의 데이터를 읽거나 쓸 수 있는 도구
- 선형대수 계산, 푸리에 변환, 난수 발생기
- 파이썬 확장과 C, C++ 코드에서 넘파이의 자료구조에 접근하고 계산 기능으로 사용할 수 있도록 하는 C API

### 1.3.2 판다스(pandas)
pandas : 구조화된 데이터나 표 형식의 데이터를 직관적이고 유연하게 다룰 수 있도록 설계된 고수준의 자료구조와 함수 제공. 판다스의 주된 자료구조는 표 형태의 행과 열이 이름을 가지는 DataFrame과 1차원 배열 객체인 Series가 있음

### 1.3.3 맷플롯립(Matplotlib)
MAtplotlib : 그래프나 2차원 데이터 시각화를 생성하는 라이브러리 

### 1.3.4 IPython과 주피터
Ipython : 대화형 파이썬 인터프리터. 전통적인 편집-컴파일-실행 방식 대신에 실행-탐색 방식을 장려하고 덕분에 터미널과 파이썬 창을 전환할 필요가 없음
주피터 : 언어에 상관없이 대화형 컴퓨팅 도구

### 1.3.5 사이파이(scipy)
SciPY : 과학 계산 영역의 여러기본 문제를 다루는 패키지 모음. 넘파이와 사이파이를 함꼐 사용하면 전통적인 과학계산 에플리케이션에서 제공하는 거의 모든 기능들을 대체 할 수 있음 
- scipy.integrate : 수치적분 루틴과 미분방정식의 풀이법
- scipy.linalg : numpy.linalg에서 제공하는 것보다 더 확장된 선형대수 루틴과 매트릭스 분해
- scipy.optimize : 함수 최적화기(optimizer)와 방정식의 근을 구하는 알고리즘
- scipy.signal : 시그널 프로세싱 도구
- scipy.sparse : 희소 행렬과 희소 선형 시스템 풀이법
- scipy.special : 감마 함수처럼 흔히 사용되는 수학 함수를 구현한 포트란 라이브러리인 SPECFUN래퍼
- scipy.ststs : 표준 연속, 이산 확률분포와 다양한 통계 테스트 그리고 좀 더 기술적인 통계도구

### 1.3.6 사이킷런
scikit-learn : 처음 개발되기 시작한 2010년 부터 파이썬 개발자를 위한 범용 머신러닝 도구로 자리잡기 시작함. 다음과 같은 하위 모듈을 포함함
- 분류 : SVM, 최근접 이웃, 랜덤 포레스트, 로지스틱 회귀 등
- 회귀 : 라소, 리지 회귀 등
- 클러스터링 : k-평균, 스팩트럼 클러스터링 등
- 차원 축소 : PCA, 특징 선택, 행렬 인수분해 등
- 모델 선택 : 격자 탐색, 교차 검증, 행렬
- 전처리 : 특징 추출, 정규화

### 1.3.7 statsmodels 
statsmodels : 스탠포드 대학의 통계학 교수 조너선 테일러의 작업을 기반으로 만들어진 통계분석 패키지. 사이킷런과 비교하면 statsmodels은 전통적인 통계와 계량경제학 알고리즘을 포함함
